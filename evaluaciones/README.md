# Evaluaciones

Esta carpeta reúne controles, pruebas, rúbricas, pautas de corrección y bancos de preguntas asociados a la asignatura de Álgebra Matricial.

## Tipos de recursos

- Controles.
- Pruebas solemnes.
- Pautas de corrección.
- Rúbricas analíticas.
- Bancos de preguntas para Canvas.
- Actividades evaluativas grupales.
- Instrumentos de autoevaluación y coevaluación.

## Criterios generales

Las evaluaciones deben favorecer:

1. Razonamiento matemático.
2. Modelamiento contextualizado.
3. Interpretación de resultados.
4. Defensa conceptual.
5. Trazabilidad del uso de IA.
6. Equidad y transparencia evaluativa.
