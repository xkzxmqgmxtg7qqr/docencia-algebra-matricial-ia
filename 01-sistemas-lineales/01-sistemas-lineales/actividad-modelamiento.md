# Actividad: Modelamiento con Sistemas de Ecuaciones Lineales

## Contexto

Una empresa de ingeniería debe distribuir recursos entre tres áreas de trabajo: soporte técnico, desarrollo de software y análisis de datos. Cada área requiere distintas horas de trabajo y distintos costos operativos.

## Objetivo

Formular, resolver e interpretar un sistema de ecuaciones lineales asociado a una situación contextualizada de ingeniería.

## Instrucciones

1. Defina las variables del problema.
2. Formule el sistema de ecuaciones.
3. Construya la matriz aumentada.
4. Resuelva el sistema mediante un método algebraico.
5. Verifique la solución obtenida.
6. Interprete los resultados en el contexto.
7. Declare el uso de IA, si corresponde.

## Producto esperado

Un informe breve que incluya:

- definición de variables;
- sistema de ecuaciones;
- matriz aumentada;
- desarrollo algebraico;
- solución;
- interpretación;
- verificación;
- bitácora de uso de IA.

## Criterios de evaluación

| Criterio | Descripción |
|---|---|
| Definición de variables | Las variables están claramente definidas y vinculadas al contexto. |
| Formulación del sistema | Las ecuaciones representan correctamente las restricciones del problema. |
| Resolución | El procedimiento algebraico es correcto y ordenado. |
| Interpretación | La solución se explica en función de la situación planteada. |
| Verificación | Se comprueba la solución obtenida. |
| Uso de IA | Se declara y verifica críticamente el apoyo de IA. |
