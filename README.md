# Docencia de Álgebra Matricial con IA

Repositorio docente orientado al diseño, organización, versionamiento y mejora continua de materiales para la enseñanza de Álgebra Matricial en carreras de ingeniería.

## Propósito del repositorio

Este proyecto busca constituir un espacio académico digital para organizar materiales de clase, guías de aprendizaje, evaluaciones, actividades de modelamiento, recursos interactivos y estrategias de integración responsable de inteligencia artificial generativa en la enseñanza de Álgebra Matricial.

## Objetivos

- Organizar materiales docentes por unidades temáticas.
- Diseñar actividades contextualizadas en ingeniería.
- Incorporar inteligencia artificial como apoyo al aprendizaje matemático.
- Mantener trazabilidad de versiones de evaluaciones y recursos.
- Generar recursos compatibles con Canvas LMS.
- Promover metodologías activas como ABP y ABR.
- Documentar experiencias de innovación educativa.

## Unidades temáticas

1. Sistemas de ecuaciones lineales.
2. Método de Gauss y Gauss-Jordan.
3. Operatoria básica de matrices.
4. Determinantes.
5. Valores y vectores propios.
6. Diagonalización.
7. Métodos numéricos: Jacobi y Gauss-Seidel.
